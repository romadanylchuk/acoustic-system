# Серверна частина (Rock Pi 4C+)

TTGO LoRa32 підключається до Rock Pi по USB/UART як комунікаційний модуль.
USB мікрофон підключається напряму до Rock Pi (станція 0).

## Алгоритми

### Тріангуляція

1. **Bearing Intersection** — перетин напрямних від станцій
2. **TDOA між станціями** — по різниці часу надходження події

### Класифікація

Нейромережева класифікація джерела:
- Людина (кроки, голос)
- Тварина
- Транспорт (колісний, гусеничний)
- Невідомо
