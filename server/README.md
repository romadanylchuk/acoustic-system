1. **Bearing Intersection** — перетин напрямних від станцій
2. **TDOA між станціями** — по різниці часу надходження події

### Класифікація

Нейромережева класифікація джерела:
- Людина (кроки, голос)
- Тварина
- Транспорт (колісний, гусеничний)
