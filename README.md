# Акустична розвідувальна система

Система акустичної розвідки для спостережного пункту (СП) у польових умовах.

## Структура проекту

```
acoustic-system/
├── docs/
│   ├── acoustic_system_context.md      # Контекст проекту (для Claude)
│   └── bom_acoustic_system_v2.xlsx     # Специфікація комплектуючих
├── hardware/
│   ├── station_battery_mount.scad      # Акумуляторний відсік + кріплення
│   ├── station_electronics_compartment.scad  # Електронний відсік
│   └── station_mic_lid.scad            # Мікрофонна кришка
├── firmware/
│   ├── station/                        # Прошивка ESP32-S3 (автономна станція)
│   └── gateway/                        # Прошивка TTGO LoRa32 (базова станція)
├── server/                             # Серверна частина (Rock Pi 4C+)
└── README.md
```

## Відновлення контексту в Claude Code

```bash
claude
```

Після запуску:
```
Прочитай docs/acoustic_system_context.md і продовжимо розробку
```

## Тестова конфігурація

```
2× Автономна станція  (ESP32-S3 + 4×INMP441 + LoRa + nRF24)
1× Базова станція     (Rock Pi 4C+ + TTGO LoRa32)
Розміщення: трикутник ~300м сторона
```
