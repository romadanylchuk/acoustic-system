# Прошивка комунікаційного модуля (TTGO LoRa32)

TTGO LoRa32 — радіоміст між автономними станціями та Rock Pi 4C+.
Мікрофон СП (USB) підключається напряму до Rock Pi.

## Підключення

### nRF24L01 (SPI)

| nRF24 | TTGO ESP32 |
|-------|------------|
| MOSI | GPIO 23 |
| MISO | GPIO 19 |
| SCK | GPIO 18 |
| CSN | GPIO 5 |
| CE | GPIO 17 |
| IRQ | GPIO 16 |

### INMP441 мікрофони СП (I2S)

Аналогічно до автономної станції — 4 мікрофони на двох I2S шинах.

### OLED дисплей (вбудований на TTGO)

| Функція | Відображення |
|---------|-------------|
| Рядок 1 | Кількість активних станцій |
| Рядок 2 | Остання подія (клас + час) |
| Рядок 3 | Рівень сигналу LoRa |
| Рядок 4 | Статус синхронізації |
