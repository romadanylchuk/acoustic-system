# hardware/schematics — Текстові нетлісти схем

Цей каталог містить текстові нетлісти у форматі, сумісному з KiCad 8, для майбутнього проектування друкованих плат акустичної розвідувальної системи.

## Файли

| Файл | Опис |
|------|------|
| `station_netlist.txt` | Нетліст автономної станції (ESP32-S3 N16R8): 4× INMP441 по двох шинах I2S, Ra-02 SX1278 LoRa (SPI), nRF24L01 PA+LNA (спільна SPI шина з LoRa), DS3231 RTC (I2C), рейки живлення від LiFePO4 |
| `gateway_netlist.txt` | Нетліст шлюзу (TTGO LoRa32): зовнішній nRF24L01 PA+LNA (SPI), вбудований SX1278 LoRa 433 MHz, вбудований OLED-дисплей SSD1306, UART до сервера Rock Pi 4C+ |

## Формат та інструменти

Файли написані в S-expression форматі `(net-list ...)` — текстовий опис з'єднань.

**Інструмент:** KiCad 8 — редактор схем і трасування ДП.

## Воркфлоу редагування схем

**Джерело правди — `.kicad_sch` файли (KiCad S-expression).**

1. Перший `.kicad_sch` генерується з нетлісту (одноразово).
2. Розкладка компонентів робиться вручну в KiCad Eeschema (одноразово).
3. Всі подальші правки вносяться **напряму в `.kicad_sch`** — Claude може редагувати цей формат як текст.
4. Перегляд — відкрити/перезавантажити файл в KiCad Eeschema.

**Чому не skidl:** skidl при перегенерації скидає ручне розкладання компонентів. Для схем зі стабільною топологією це зайвий overhead.

```
Описати помилку → Claude править .kicad_sch → перевідкрити в KiCad → layout збережено
```

## Параметри запису (прототип)

- Частота дискретизації: **32 000 Гц / 16-bit**, 4 канали
- Кільцевий буфер: 32000 × 4 кан. × 2 байт × 15 с = **3.84 МБ** (в межах 8 МБ PSRAM модуля N16R8)
- Смуга: 16 кГц — покриває акустичні сигнатури БПЛА

## Серійний зразок (цільова апаратна платформа)

Серійна версія передбачає перехід на підвищені характеристики:

| Параметр | Прототип | Серійний зразок |
|----------|----------|-----------------|
| МК | ESP32-S3 N16R8 | **STM32H743** |
| Мікрофони | INMP441 | **ICS-43434** (±0.5° фазова узгодженість) |
| Частота дискретизації | 32 000 Гц / 16-bit | **44 100 Гц / 24-bit** |
| Смуга | 16 кГц | 20 кГц |
| Буфер | 3.84 МБ | ~7.9 МБ (потребує зовнішнього SDRAM або ≤10 с буфера) |

Нетлісти для серійної версії будуть додані окремо після уточнення схемотехніки STM32H743.
